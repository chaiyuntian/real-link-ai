# Issue Map

The first wave maps GitHub issues to playable folders like this:

- Issue #1 `Rhythm Cut` -> `gameplay/issue-001-rhythm-cut/`
- Issue #2 `Freeze Frame Hunter` -> `gameplay/issue-002-freeze-frame-hunter/`
- Issue #3 `Director Split` -> `gameplay/issue-003-director-split/`
- Issue #4 `Glitch Chase` -> `gameplay/issue-004-glitch-chase/`
- Issue #5 `Remix Runner` -> `gameplay/issue-005-remix-runner/`

All five prototypes are standalone and can be launched from the repo root index page.
